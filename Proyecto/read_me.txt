Proyecto Procesamiento de Imágenes y Video

Base de datos: La base de datos fue creada manualmentea través de imágenes de caras 
tomadas de teléfonos celulares y computadores. Actualmente se cuentan con 11 personas, 
para la próxima entrega se piensa utilizar fotos de caras de bases de datos online. Para 
aumentar la base de datos se utiliza el algoritmo MaskTheFace, el cual dejo a continuación
el link del proyecto de GitHub.

MaskTheFace: https://github.com/aqeelanwar/MaskTheFace
